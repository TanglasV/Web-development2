These are the features that has been implemented in this website

  1. Current page is highlighted in the navigation bar by an overline in Home, Register, Order, and Features every page.
  2. Billing address in Order page can be filled automatically by clicking the 'Same as delivery address' button right below it, provided that delivery address is filled up.
  3. Card number field in Order page is dynamically length-limited based on the selected card type. MasterCard and Visa holders can input 16 digits and American Express holders can input 15 digits - which corresponds to actual card number length.
